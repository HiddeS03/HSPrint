# Assets Folder

Plaats hier je logo bestanden voor de HSPrint applicatie:

- `icon.ico` - Windows icoon voor de Config Tool applicatie
- `installer-icon.ico` - MSI installer icoon
- `logo.png` - PNG logo voor about dialogs (optioneel)

Zie `LOGO-GUIDE.md` in de root van het project voor gedetailleerde instructies over het toevoegen van logo's.
